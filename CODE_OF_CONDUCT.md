# Code of Conduct

This project adopts the Contributor Covenant Code of Conduct.

## Our Standards
- Be respectful and constructive.
- Assume positive intent.
- Focus on technical outcomes and collaboration.

## Enforcement
Project maintainers may remove or edit comments, commits, code, issues, and other contributions that are not aligned with this Code of Conduct.

## Reporting
To report unacceptable behavior, open a private security-style issue or contact maintainers through repository channels.
